## Standardized Project Information

To improve consistency and help users make informed decisions, all project entries in this list follow a [standardized format](https://github.com/ranjithrajv/awesome-aquarium-tech/blob/main/docs/PROJECT_TEMPLATE.md) that includes essential information like platform requirements, difficulty level, hardware requirements, and key features. This makes it easier to compare projects and understand what's involved before starting implementation.

## AI/ML-Powered Solutions

- [Smart Fish Tank Monitoring System](https://github.com/IT21179280/SMART-FISH-TANK-MONITORING-SYSTEM) - IoT-based system using ESP32-CAM with custom-trained YOLOv8 models for fish species identification, pregnant fish detection, and automated environment control.

- [Smart Aquarium IoT Analytics Platform](https://github.com/YasinduKaveesha/smart-aquarium-iot-analytics-platform) - Adaptive AI-driven IoT monitoring platform with SARIMA forecasting, Isolation Forest anomaly detection, and fuzzy Water Quality Index scoring.

## Aquarium Lighting Control

- [DeepGlow](https://github.com/kabroxiko/DeepGlow) - Standalone ESP-IDF based aquarium LED controller for ESP32-family boards with real-time control and smooth transitions.

- [AquaControl32](https://github.com/CelliesProjects/aquacontrol32-pio) - 5-channel LED aquarium controller with gradual sunrise/sunset simulation, moon phase cycle, and web interface.

## Aquascaping Tools

- [Maya's Aquarium Maker 3D](https://github.com/m4ym4y/mayas-aquarium-3d) - Virtual fish tank editor built with Three.js and react-three-fiber for designing aquarium layouts in 3D.

- [Aquarium Builder](https://github.com/randomnewb/aquarium-builder) - Tool for hobbyists to plan and organize aquarium components - create, view, and modify tank setups before building.

## Aquatic Plant Care Systems

- [ESP-aqua-CO2](https://github.com/Jbsco/ESP-aqua-CO2) - Low-cost CO2 control system for automated aquariums using LilyGO T-Display-S3 ESP32, maintaining target pH levels.

- [DIY my Dose](https://github.com/diymoica/DmDose) - Universal smart dosing controller for reef aquariums - controls up to 4 peristaltic pumps with Home Assistant + ESP32 (ESPHome).

## Biological Filtration Systems

- [THIR Aquarium Roll Filter](https://github.com/DeeEmm/Thir) - Open source DIY automatic roll filter for mechanical filtration - build-it-yourself with laser-cut perspex and off-the-shelf filter media.

## Emergency and Safety Systems

- [ESPHome Water Leak Detector](https://github.com/mrahmadt/ESPHome-Water-Leak-Detector) - DIY water leak detection system built with ESPHome and Home Assistant for detecting water leaks around aquariums.

- [AquariumController](https://github.com/disgustipated/AquariumController) - ESP8266-based aquarium overflow prevention using ultrasonic sensors (JSN-SR04T) and DS18B20 temperature probe for water level and temperature monitoring.

- [reefbeatEnergyBackup](https://github.com/Elwinmage/reefbeatEnergyBackup) - Control Reefbeat devices during power outages with automated backup system management.

## Energy Management

- [ESP32 Power Monitor](https://github.com/Maxim-szh/ESP32-Power-Monitor-WebServer) - ESP32-based power monitoring using SCT-013 current sensors with real-time data visualization via web interface.

- [Smart Aquarium V3.1](https://github.com/desiFish/Smart-Aquarium-V3.1) - ESP8266-based aquarium controller for managing 4 relay outputs with manual, scheduled, timer, and toggle modes.

## Comprehensive Controllers

- [Aquareo](https://github.com/fnandes/aquareo) - Open-source ESP32-based aquarium controller for monitoring and automating basic aquarium needs.

- [aquaPI](https://github.com/schwabix-1311/aquaPI) - Modular aquarium controller for Raspberry Pi with web interface for monitoring and control.

- [Marine Assistant](https://github.com/marine-assistant/Marineassistant) - Smart aquarium controller designed for reef tanks with local Home Assistant integration, no cloud required.

- [OctoRelay](https://github.com/borisbu/OctoRelay) - OctoPrint/OctoPi plugin to control relays, allowing users to control GPIO pins of Raspberry Pi for switching relays and indicating their states, with support for up to 8 relays.

- [Orca](https://github.com/mstelz/orca) - Raspberry Pi-based aquarium controller with cloud-based website for monitoring and management.

- [reef-pi](https://github.com/reef-pi/reef-pi) - An open-source reef tank controller based on Raspberry Pi with modular design for affordable DIY automation.

## Feeding Automation

- [fishfeeder](https://github.com/ColoMAX/fishfeeder) - Standalone automatic fish feeder based on ESP8266 with ESPHome, featuring 3D-printed Archimedes screw dispenser.

- [iotFeeder](https://github.com/reivaxy/iotFeeder) - WiFi-enabled fish feeder using ESP8266 and stepper motor with notifications when empty or on power failure.

## Hardware Projects

- [Aqua-Controller](https://github.com/atanasyanew/aqua-app) - ESP8266-based fish tank automation using Mongoose OS and Node-RED for remote monitoring and control.

- [Aquaero](https://github.com/shred/pyquaero) - Python library and web service for accessing Aquaero fan and pump controllers by Aqua Computer GmbH & Co. KG. Community open source project not endorsed by or affiliated with Aqua Computer. Based on reverse engineering.

- [Aquariumduino](https://github.com/FernandoGarcia/FerduinoAquariumController) - Open-source aquarium controller project based on the Arduino platform, featuring temperature, pH, ORP, and density monitoring/control, automatic top-off, partial water change automation, LED light control, cloud/lightning simulation, and automatic feeder with web interface.

- [Buce LED Controller](https://github.com/borneo-iot/borneo) - Compact ESP32-C3 based controller supporting 6 independent PWM channels for aquarium lighting.

- [EcolibriumReef](https://github.com/PracticalHomeApp/EcolibriumReef-Arduino) - DIY Arduino-based aquarium controller with Android companion app.

- [JeeLabs](https://github.com/jeelabs) - Organization focused on embedded systems, IoT, and Arduino-related projects. Most popular repository is esp-link (esp8266 wifi-serial bridge) with nearly 3,000 stars, followed by jeelib (Arduino library) with 507 stars.

- [OpenAquarium](https://github.com/balub/open-aquarium-project) - Collection of FOSS tools/configs along with hardware and software for aquarium management. Contains configs/guides to FOSS tools and software and hardware developed for managing aquariums.

- [OpenSprinkler](https://github.com/OpenSprinkler/OpenSprinkler-Firmware) - Unified firmware for OpenSprinkler, OpenSprinkler Pi, and Linux-based OpenSprinkler. Supports multiple platforms and is actively maintained with substantial community following.

- [Reef Angel](https://reefangel.com/) - Open-source aquarium controller built on the Arduino platform, designed for reef aquariums.

- [Smart Aquarium Project](https://github.com/yashkurkure/Smart-Aquarium-Project) - IoT-based aquarium automation with timed feeding, temperature monitoring, water level detection, and light control.

- [TankBuilder](https://github.com/awitwicki/openTank) - Open source tank platform that includes implementations for ESP32, Raspberry Pi, and Arduino. Uses 3D printed parts, plastic tracks, DC motors, L298N motor driver, and supports various microcontrollers.

## Monitoring and Automation Systems

- [AquaMonitor](https://github.com/reivaxy/aquaMonitor) - Recifal aquarium monitoring with Arduino, alerts and settings by SMS and webApp, and statistics database. Monitors light level, temperature, water level, main power failure, and sends SMS alerts when configurable thresholds are reached.

- [AquaPi (aquapi)](https://github.com/TheRealFalseReality/aquapi) - Aquarium monitoring and automation system using ESPHome and Home Assistant for smart home integration.

- [Domoticz](https://github.com/domoticz/domoticz) - Open source Home Automation System that monitors and configures various devices like lights, switches, sensors/meters (Temperature, Rain, Wind, UV, Electricity, Gas, Water, etc.) with notifications, cross-platform support, HTML5 web frontend, and extended logging.

- [Fish Tank Monitor and Control](https://github.com/rjsears/Fish_Tank_Monitor_and_Control) - Comprehensive monitoring and management system for fish tanks using Python/Flask with sensor integration.

- [OpenHAB](https://github.com/openhab/openhab-core) - Core framework of openHAB - Contains core bundles of the openHAB runtime. Not a product itself, but a framework to build solutions on top of, serving as the foundation for the main openHAB distribution.

## Specialized Tools

- [AQUASIM](https://github.com/jsta/AQUASIM) - Computer Program for the Identification and Simulation of Aquatic Systems. This is a mirror of the original AQUASIM software from EAWAG (Swiss Federal Institute of Aquatic Science and Technology).

- [FishPi](https://github.com/alicegodden/fishpi) - piRNA TE analysis tool - A Python-powered solution for piRNA sequence analysis that identifies complementary transposable element (TE) sequences, with focus on fish genomics.

- [HYDROS App Framework](https://www.hydros.com/) - Open-source framework for connecting aquarium equipment to cloud services for remote monitoring.

- [Rabdomante](https://github.com/molok/rabdomante) - Brewing Water Chemistry Calculator that helps users find the optimal mix of waters and salts to match a target water profile. Calculates the distance from the target profile based on Calcium (Ca), Magnesium (Mg), Sodium (Na), Sulfate (SO4), Chloride (Cl), and Bicarbonates (HCO3).

- [Aquarium Tank Monitoring System](https://github.com/jsramesh1990/Aquarium-Tank-Monitoring) - Multi-board embedded project for monitoring critical water parameters (temperature, pH, TDS) supporting Arduino UNO, Mega, ESP32, ESP8266, STM32, and Pi Pico.

- [aqua-simulation](https://github.com/ranjithrajv/aqua-simulation) - Aquarium Tank Simulator - A web-based tool that provides 3D visualization of aquarium tanks and calculates glass thickness recommendations based on tank dimensions.

## Water Chemistry Management

- [Open Dosing Pump](https://github.com/inventmarine/open-dosing-pump) - Open source, DIY, easy-to-assemble peristaltic dosing pump for precise chemical addition - no soldering required, 4 pumps with Bluetooth or USB.

- [Aquamonitor](https://github.com/aquamonitor/Aquamonitor) - Automated RO/DI auto top-off (ATO) and auto water change system for reef tanks using Raspberry Pi - under $200.

## Water Testing & Analysis Tools

- [Smart Fish Tank](https://github.com/Spenc3rB/Smart-Fish-Tank) - ESP32-based environmental monitoring system for TDS, temperature, RGB LED control, stepper motor feeder, and email alerts.

- [Web-Based Aquarium Automation](https://github.com/akhdanfarrosrenoldi/web-realtime-aquarium-automation) - ESP32-based real-time aquarium monitoring system tracking temperature and TDS/EC with ThingSpeak integration.
