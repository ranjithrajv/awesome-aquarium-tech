# Tagging System for Aquarium Technology Projects

This document outlines the comprehensive tagging system used to categorize projects in the awesome-aquarium-tech repository for easier discovery and classification.

## Tag Categories

### Platform Tags
- `arduino` - Arduino-based projects
- `raspberry-pi` - Raspberry Pi-based projects
- `esp32` - ESP32-based projects
- `esp8266` - ESP8266-based projects
- `pc-based` - Computer-based systems
- `mobile-app` - Mobile applications

### Function Tags
- `monitoring` - Monitoring systems
- `automation` - Automation systems
- `control` - Control systems
- `dosing` - Dosing systems
- `lighting` - Lighting systems
- `feeding` - Feeding systems
- `filtration` - Filtration systems
- `safety` - Safety systems

### Difficulty Tags
- `beginner` - Suitable for beginners
- `intermediate` - Moderate skill level required
- `advanced` - Advanced knowledge required
- `expert` - Expert level projects

### Hardware Tags
- `sensors` - Sensor-based projects
- `actuators` - Projects with moving parts
- `electronics` - Electronic projects
- `3d-printing` - 3D printing involved
- `mechanical` - Mechanical components

### Communication Tags
- `wifi` - WiFi connectivity
- `bluetooth` - Bluetooth connectivity
- `mqtt` - MQTT protocol
- `serial` - Serial communication
- `ethernet` - Ethernet connectivity

### Application Tags
- `freshwater` - Freshwater applications
- `saltwater` - Saltwater applications
- `planted-tank` - Planted tank applications
- `reef-tank` - Reef tank applications
- `nano-tank` - Nano tank applications

## Tagging Convention

Tags should be applied in the following format:
```
Tags: #platform #function #difficulty #hardware #communication #application
```

## Example Tagging

**Example 1**: A simple temperature monitoring project using Arduino
- Tags: #arduino #monitoring #beginner #sensors #serial #freshwater

**Example 2**: A complex reef automation system using Raspberry Pi
- Tags: #raspberry-pi #automation #advanced #sensors #actuators #wifi #mqtt #reef-tank

## Benefits of Tagging System

1. **Improved Discoverability**: Users can quickly find projects matching their interests
2. **Better Filtering**: Facilitates filtering based on multiple criteria
3. **Cross-Referencing**: Enables linking of related projects
4. **Progression Tracking**: Helps users follow a learning path
5. **Resource Optimization**: Allows for better resource allocation and planning

## Implementation Guidelines

1. Apply 3-6 relevant tags per project
2. Use the most specific tag available
3. Maintain consistency in tag usage
4. Update tags as projects evolve
5. Regular tag system review and refinement

## Tag Maintenance

The tag system should be reviewed quarterly to:
- Add new relevant tags
- Deprecate unused tags
- Consolidate overlapping tags
- Improve tag descriptions
- Enhance tag utility

This tagging system enables users to efficiently navigate the diverse range of aquarium technology projects based on their specific needs, skill levels, and interests.
